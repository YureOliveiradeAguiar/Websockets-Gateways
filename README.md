A project study about websockets and gateways, using socket.io and demonstrated via a live list.<br><br>
Includes CRUD with typeORM, list with pagination, animated input fields and some advanced UI components.
<h4>Dependencies</h4>
(nest) (for websocket) npm install @nestjs/websockets @nestjs/platform-socket.io<br>
(nest) (fot typeORM and database) npm install @nestjs/typeorm typeorm sqlite3<br>
(angular) (for websocket) npm install socket.io-client
